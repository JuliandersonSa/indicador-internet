📡 Indicador de Qualidade da Conexão Residencial (IQCR)

Este projeto demonstra a construção de um indicador de desempenho a partir de dados reais coletados com um script em Python. O objetivo é analisar a qualidade da internet contratada (200 Mbps) por meio de testes automatizados.

🔢 Fórmula do Indicador

IQCR = (Velocidade média de download real / Velocidade contratada) × 100

🔍 Sobre a coleta de dados

A coleta foi realizada com o speedtest-cli rodando automaticamente a cada 9 minutos usando agendamento via crontab. Foram monitorados os parâmetros: download (Mbps), upload (Mbps) e ping (ms).
